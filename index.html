比较ONT miniON 及BGI Cyclone WT-02

发表于 2025-04-10

由于业务需要，需要对华大智造的 CycloneSEQ-WT02 和 Oxford Nanopore Technologies 的 miniON 相关参数进行比较。
官网链接：https://www.mgi-tech.com/products/instruments_info/73/
https://nanoporetech.com/products/sequence/minion

1. 技术原理

  • CycloneSEQ-WT02: 采用纳米孔测序技术，通过马达蛋白解旋DNA单链，使其通过纳米孔蛋白，基于电流波动实时解析碱基序列。使用自主研发的纳米孔蛋白和马达蛋白（源于深海细菌样本），并结合深度学习优化的碱基识别算法。
  • MinION: 同样基于纳米孔测序，利用蛋白纳米孔嵌入高电阻膜，通过DNA或RNA分子穿过纳米孔时引起的电流变化识别碱基序列。核心技术依赖于Oxford Nanopore的专利蛋白（如CsgG纳米孔）和专用集成电路（ASIC）。

比较: 两者原理相似，但CycloneSEQ-WT02强调自主研发的蛋白和算法，而MinION依赖成熟的专利生态和技术优化。

2. 硬件设计

  • CycloneSEQ-WT02:
    • 双芯片架构，每个流动池（flow cell）最多支持4096个纳米孔蛋白。
    • 支持双流动池独立运行，增加灵活性和通量。
    • 尺寸为鞋盒大小，便携性中等。
  • MinION:
    • 单流动池设计，每个流动池最多512个通道，每个通道可控制4个纳米孔（理论最大2048个纳米孔）。
    • 超小型便携式设计（重量约100克），通过USB连接笔记本电脑运行。

比较: CycloneSEQ-WT02在硬件上更倾向于高通量和多任务处理，而MinION以便携性和简洁性著称。

3. 测序通量

  • CycloneSEQ-WT02:
    • 单流动池最大实验室产出50 Gb，双流动池可达100 Gb。
    • 通量随样本类型和测序时间变化（10分钟至107小时灵活设置）。
  • MinION:
    • 单次运行最大产出约10-20 Gb（与流动池版本有关，官网说明最大为48 Gb）。
    • 运行时间通常为48-72小时，但可随时停止。

比较: CycloneSEQ-WT02在通量上高于MinION，特别是双流动池测序模式，尤其适合需要高数据量的场景，而MinION更适合小型或快速实验。

4. 读长

  • CycloneSEQ-WT02:
    • 支持100 bp至Mbp级读长，N50通常为30-40 kb，超长读长模式下可达1.1 Mbp。
    • 读长与样本制备密切相关。
  • MinION:
    • 支持短至数百bp至超长读长（记录超过4 Mbp），N50一般在20-50 kb。
    • 可直接测序DNA和RNA，擅长超长片段分析。

比较: 两者读长能力相近，均适合复杂基因组拼接，但MinION在超长读长记录上稍占优势。

5. 测序速度

  • CycloneSEQ-WT02:
    • 平均速度约350-420 bp/s（每秒碱基）。10 Mb/min
  • MinION:
    • 速度约450 bp/s（最新版本数据），实时性强。

比较: MinION在单分子测序速度上略快，但差距不大。

6. 准确性

  • CycloneSEQ-WT02:
    • 原始读长准确率（raw read accuracy）众数为97%（Q15），范围94-97%。
    • 一致性准确率（consensus accuracy）在40x深度下达99.99%（Q40）。
  • MinION:
    • 原始读长准确率约92-97%（视版本和碱基调用模型而定，最新R10.4流动池接近97%）。
    • 一致性准确率在高深度下可达99.9%以上（Q30-Q40）。

比较: CycloneSEQ-WT02原始准确率与MinION最新版本接近，但MinION得益于多年的算法优化，在一致性准确率上表现更稳定。

7. 应用场景

  • CycloneSEQ-WT02:
    • 适用于扩增子测序、宏基因组学、小型基因组全基因组测序、低深度人类全基因组测序、全转录组测序。
    • 强调复杂序列（如重复区域）的解析能力。
  • MinION:
    • 广泛应用于微生物鉴定、现场快速检测（如疫情监控）、全长转录组分析、表观遗传学研究。
    • 便携性使其在野外和临床场景中表现出色。

比较: CycloneSEQ-WT02更适合实验室高通量需求，MinION则在便携和实时检测中占据优势。

8. 成本与可用性

  • CycloneSEQ-WT02:
    • 2024年9月全球发布，中国区首发，海外市场逐步推出。
    • 华大智造倾向于提供性价比高的解决方案。
  • MinION:
    • 已商业化多年，入门套装约1000美元，流动池单价约500-900美元。
    • 全球供应稳定，生态系统成熟。

比较: MinION在市场成熟度和配件支持上领先，CycloneSEQ-WT02作为新产品可能在初期成本更具竞争力，但供应链尚待完善。

9. 优缺点总结

  • CycloneSEQ-WT02:
    • 优点: 高通量、灵活的双芯片设计、自主技术减少专利依赖。
    • 缺点: 新产品，生态系统和用户经验较少，全球可用性有限。
  • MinION:
    • 优点: 超便携、成熟的技术支持、广泛的社区和应用案例。
    • 缺点: 通量较低，成本随大规模使用可能上升。

结论

  • 需要高通量和实验室级应用（如大基因组组装或多样本处理），CycloneSEQ-WT02可能是更好的选择。
  • 如果需要便携性、实时性和野外快速检测，MinION是目前更成熟

ONT miniON

发表于 2025-04-10

一、技术原理与核心机制

MinION 是基于纳米孔测序技术的便携式第三代测序仪，其核心原理是通过纳米孔蛋白和马达蛋白的协同作用，实现DNA/RNA分子的解旋和通过纳米孔，从而解析碱基序列。

  1. 纳米孔传感技术
    单链DNA或RNA分子在电场驱动下，通过嵌入高电阻膜的纳米孔蛋白（如CsgG）通道。不同碱基（A、T、C、G）或修饰碱基通过纳米孔时，会引起特征性的离子电流变化。专用集成电路（ASIC）实时检测这些电流信号，直接解析碱基序列。该技术无需PCR扩增或荧光标记，可保留核酸分子的天然状态。

  2. 长读长与实时性
    支持从数百bp到超长读长（记录超过4 Mbp）的测序能力，能够跨越复杂重复序列和结构变异区域，提升基因组组装完整性。测序过程实时生成数据，用户可随时停止运行，灵活性高。

  3. 多类型碱基检测
    不仅能识别标准碱基，还可直接检测甲基化（如5mC、6mA）等表观修饰位点，支持表观遗传学和RNA修饰研究。

  4. 核心技术组成

    • 蛋白质: 使用经过优化的马达蛋白（如E8或E8.1）和纳米孔蛋白（如CsgG），马达蛋白控制DNA/RNA的解旋和通过速度，纳米孔蛋白提供高信噪比的电流信号。
    • 流动池: 单流动池含512个通道，每个通道最多控制4个纳米孔（理论最大2048个纳米孔），通过微阵列设计实现高效测序。
    • 化学反应系统: 利用离子缓冲体系和电场控制马达蛋白活性，确保稳定的分子通过速率。
    • 算法: 基于深度学习的碱基调用算法（如Guppy），通过训练优化信号解码，原始准确率可达97%（R10.4版本）。

二、核心优势

  • 全读长: 支持从短片段到超长DNA/RNA分子的完整测序，解析复杂基因组结构。
  • 直接测序: 无需扩增或化学修饰，直接测序天然核酸分子，保留原始遗传和表观信息。
  • 实时检测: 数据实时生成并分析，缩短从样本到结果的时间。
  • 仪器便携: 体积小巧（100克），通过USB连接笔记本电脑运行，适合现场测序。
  • 简单快速: 文库制备流程简便，最快1小时内完成从样本到测序的全过程。

三、主要应用领域

  1. 科研领域

    • 基因组组装: 适用于微生物、动植物基因组的高质量从头组装，尤其擅长解析重复区域。
    • 表观遗传学研究: 直接检测DNA甲基化和RNA修饰，提供表观组学数据。
    • 环境与生态研究: 分析微生物群落多样性，如海洋、土壤微生物研究。

  2. 临床医学

    • 病原体快速检测: 在疫情中（如埃博拉、COVID-19）实现病毒快速鉴定和变异追踪。
    • 肿瘤基因组学: 检测结构变异和融合基因，支持癌症研究。
    • 遗传病筛查: 解析长片段变异，用于罕见病诊断。

  3. 公共卫生与疾控

    • 病毒进化追踪: 实时测序新冠病毒株，支持变异株监测。
    • 突发疫情应对: 在野外或资源有限地区快速溯源病原体。

  4. 农业与生态

    • 作物育种: 解析复杂基因组，助力农业改良。
    • 物种鉴定: 用于生物多样性监测和食品溯源。

  5. 法医学与公安

    • 个体识别: 分析长STR区域，提升DNA分型精度。

  2. 微生物学研究

CycloneSEQ-WT02

发表于 2025-04-09 更新于 2025-04-10

CycloneSEQ平台的测序过程中，连接马达蛋白的 DNA 文库分子在电场力作用下，接近纳米孔道蛋白入口。马达蛋白在接近纳米孔入口时被激活，稳定且快速地解旋DNA，使文库以单链形式通过纳米孔道蛋白。由于不同DNA碱基及其排列对电流的阻碍程度不同，DNA通过纳米孔时会产生电流波动。测序芯片上的通道传感器捕捉这些电流信号并由基于机器学习的算法（basecalling）对电流信号进行解析，从而实现实时获取待测样本的碱基序列信息。

CycloneSEQ-WT02 是华大智造（MGI）自主研发的第三代纳米孔基因测序仪，基于纳米孔测序技术，通过以下核心机制实现高效测序：

  1. 纳米孔传感技术
    测序时，单链DNA/RNA分子在电场驱动下通过纳米孔蛋白通道。不同碱基（A/T/C/G）通过纳米孔时会引起特征性电流变化，通过实时检测电流信号差异，直接读取碱基序列。该技术无需PCR扩增或荧光标记，保留天然核酸分子状态。
  2. 长读长与高通量
    支持 100 bp 至 Mbp 级别 的超长读长，能够跨越复杂重复序列和结构变异区域，显著提升基因组组装完整度。单芯片通量达 50Gb，结合双芯片架构设计，可灵活选择通量以适应不同应用场景。
  3. 实时测序与快速分析
    测序速度高达 350~420 nt/s，最快 10分钟 内完成100 Mb数据的读取，5分钟内获取100 Kb长片段数据。测序过程中可实时生成FASTQ格式结果，支持“随启随停”的灵活操作模式，降低实验成本。
  4. 多类型碱基检测
    不仅能识别天然碱基，还可检测甲基化等表观修饰位点，为表观基因组学研究提供支持。

蛋白质工程、流动池设计、化学反应和碱基调用算法是 CycloneSEQ 纳米孔测序技术的主要组成部分。纳米孔测序的技术竞争目前集中在这四个组成部分上：

◾蛋白质：在分析从马里亚纳海沟等极端环境中收集的5亿多深海宏基因组数据库时发现，深海中与基因操控相关的蛋白质丰度较高，结构多样性较大。其中有一种强健的核酸操控蛋白，其核心功能区结构与大多数已知马达蛋白有明显区别，具有更强的解旋能力，且为热液区来源，热稳定性高；另一类是由均聚物自组装形成的致密结构孔道蛋白，孔径较小，具有结构刚性强、测序噪音低、质量高等特点。通过蛋白质工程改造，马达蛋白被设计成测序速度快、稳定性强、可长时间维持测序，而孔道蛋白则具有捕获能力强、稳定性高、可持续测序、信噪比高等特点，构成了CycloneSEQ的核心蛋白基础。

◾流动池：单个流动池可拥有多达4096个纳米孔蛋白传感器，以高密度阵列排列。传感器微腔直径和间距经过精细优化，可实现最佳膜孔贴合，实现超低测序噪音。扩大的微腔体积可实现超长测序时间。Bio Micro的集成

采用专用集成电路（ASIC）的生物微机电系统（BioMEMS）能够精确检测皮安级电流，提供实时稳定地读取核酸序列信息。

◾化学反应系统：纳米孔附近的化学环境对于获得高质量的测序数据至关重要。与传统的大规模化学反应不同，CycloneSEQ 开发了一种新型化学反应，利用每个孔的电子流动池来控制离子浓度。通过屏蔽缓冲液中的所有镁离子，只利用电压将它们释放到孔附近很小的区域，ATP-Mg 被激活，从而控制马达蛋白的活性。

◾算法：测序解码算法融合了自动语音识别（ASR）领域最先进的（SOTA）解决方案，利用经过大量数据训练的深度神经网络，从所有可能的组合中识别出最佳序列并进行精确解码。CycloneSEQ 自主研发了碱基调用算法模型，该算法模型基于海量数据，经过大规模分布式训练，准确率高达97%，且能够实现多flow cell的实时测序解码，性能稳健

二、核心优势

  • 全读长 纳米孔测序技术能够捕获并解析从短片段到超长DNA分子，提供无断裂的全长序列信息。
  • 直接测序 纳米孔测序技术可以直接对未经修饰或扩增的核酸分子进行测序，保留原始遗传信息，具有多组学拓展潜力。
  • 实时检测 纳米孔测序支持即时数据分析，显著减少从样本采集到结果解读的时间间隔。
  • 仪器便携 仪器小巧，测序设备易于携带，极大地扩展了现场测序的应用范围。
  • 简单快速 实验流程简化，能迅速完成从样品制备到最终数据分析的全过程。

三、主要应用领域

  1. 科研领域
    • 基因组组装：用于动植物基因组、微生物基因组的高质量从头组装，尤其适合复杂基因组（如酵母、深海细菌）。
    • 表观遗传学研究：检测DNA甲基化、RNA修饰等表观遗传标记。
    • 环境与生态研究：分析微生物群落多样性及功能基因，如深海细菌耐压机制研究。
  2. 临床医学
    • 病原体快速检测：在塞拉利昂疟疾监测中，6小时内完成疟原虫抗性基因突变鉴定，支持传染病防控。
    • 肿瘤基因组学：结合短读长测序（如DNBSEQ-G50），实现肿瘤突变和融合基因的全长分析。
    • 出生缺陷防控：用于遗传病筛查及罕见变异检测。
  3. 公共卫生与疾控
    • 病毒进化追踪：成功解析HIV病毒近全长基因组，鉴定新CRF亚型分支。
    • 突发疫情应对：实时测序能力支持病原体快速溯源，如新冠变异株监测。
  4. 农业与生态
    • 作物育种：应用于农业基因组研究，助力多年生作物（如多年生稻）的基因优化与品种改良。
    • 物种鉴定：通过动物源性食品的基因溯源，保障食品安全（如华大鉴正的种属鉴定系统）。
  5. 法医学与公安
    • 个体识别：利用长读长优势解析复杂STR区域，提升法医DNA分型精度。

四、典型案例

  1. 国际合作项目
    中国疾控中心与华大合作，在塞拉利昂利用WT02结合DNBSEQ-G50，完成疟原虫抗性基因的长短读长联合分析，显著提升当地疟疾分子监测能力。
  2. 深海生物学研究
    自然资源部第四海洋研究所结合WT02与DNBSEQ-G99，解析深海细菌耐压机制，揭示氧化三甲胺（TMAO）的作用通路。
  3. 临床联合应用
    中国疾控中心通过WT02与MGISEQ-2000联用，实现HIV病毒近全长基因组的精准组装，推动病毒进化研究3。

五、技术发展前景

  • 智能化整合：华大基因推出的生成式生物智能（GBI ALL）与iGeneT Pro系统，将WT02与AI分析结合，实现从测序到报告的全流程自动化。
  • 全球化布局：截至2024年，华大智造测序仪全球装机量近4000台，覆盖100多个国家，WT02的便携性使其在资源有限地区（如非洲）的应用潜力巨大。
  • 多组学融合：结合时空组学（Stereo-seq）与单细胞技术，推动生命科学从“序列解析”向“时空动态”研究的跨越。
